## Суть системы
Суть системы предварительной загрузки заключается в оптимизации, а так же, опционально, ограничении доступной карты для игроков путём создания процедурно сгенерированной области определённого размера. Это позволяет не добавлять `BiomeComponent` на карты, позволяя сохранить ресурсы сервера и не нагружать его регулярно обновляющимися 4+ планетами.
Система существует ТОЛЬКО на сервере.

## Принцип работы
Система, будь то по команде, либо вызову метода сторонней системой, генерирует тайл планеты на каждый отсутствующий тайл в радиусе карты. Для загрузки области используется либо команда `planet-limited`, либо метод `BiomePreloadSystem.CreateAndGenerate`.

### Использование команды
Краткий вид команды:
`planet-limited <mapId> <biome template> <radius X> <radius Y> <limitingEnt> <offset X> <offset Y>`

Пояснение пунктов:
- `<mapId>` - айди целевой карты.
- `<biome template>` - прототип биома планеты.
- `<radius X>` и `<radius Y>` - размеры области. Заметьте, что итоговый размер будет в 2 раза (+ 1 тайл) больше, чем указано.
- `<limitingEnt>` - сущность, которая будет ограничивать сгенерированную зону. Может быть null.
- `<offset X>` и `<offset Y>` - центральная позиция, относительно которой будет генерироваться зона.

### Использование метода
Метод, точнее его аргументы выглядят следующим образом:
```cs
public void CreateAndGenerate(EntityUid mapUid,
                              Vector2i scale,
                              Vector2i offset,
                              ProtoId<BiomeTemplatePrototype> biomeTemplate,
                              string? limitingEntity = null,
                              int? seed = null,
                              Color? mapLight = null)
```

Пояснение аргументов:
- `EntityUid mapUid` - сущность целевой карты
- `Vector2i scale` - размеры области. Заметьте, что итоговый размер будет в 2 раза (+ 1 тайл) больше, чем указано.
- `Vector2i offset` - центральная позиция, относительно которой будет генерироваться зона.
- `ProtoId<BiomeTemplatePrototype> biomeTemplate` - прототип биома планеты.
- `string? limitingEntity = null` - сущность, которая будет ограничивать сгенерированную зону. Можно не указывать.
- `int? seed = null` - Используемый для генерации сид. Можно не указывать.
- `Color? mapLight = null` - Освещение карты. Можно не указывать.
